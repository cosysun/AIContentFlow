## OpenClaw 是什么？（30 秒版本）
如果你没听说过 OpenClaw，一句话解释：它是一个开源 AI Agent，能在你的服务器上 7×24 运行，通过 Telegram / 微信 / 钉钉等渠道接受你的指令，并真正执行任务——不只是回答问题。
和 ChatGPT 最大的区别？ChatGPT 是「顾问」，你问它，它说，然后你去做。OpenClaw 是「执行者」，你说，它直接帮你做完。
它通过 MCP（Model Context Protocol） 协议连接各种外部工具，Notion 就是其中一个。配置好之后，你对着 Telegram 说一句话，OpenClaw 就能在 Notion 里新建页面、搜索内容、更新记录——全自动。

## 常见问题
Q：不会编程，能配置吗？
A：不需要编程，只要和OpenClaw对话，让它做
Q：Notion 免费版够用吗？
A：个人使用完全够。免费版的 API 调用限制是每秒 3 次请求，对个人知识管理来说完全不是瓶颈。
Q：数据安全吗？
A：OpenClaw 支持本地部署，你的数据不会经过第三方服务器。Notion Token 存在你自己服务器的 .env 文件里，只要服务器安全，数据就安全。
Q：服务器需要一直开着吗？
A：是的，OpenClaw 需要 7×24 运行。推荐用云服务器，阿里云/腾讯云轻量服务器每月约 30-50 元，性价比很高。
