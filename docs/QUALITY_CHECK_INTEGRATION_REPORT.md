## 💡 最佳实践

### 何时执行质检？

**必须执行**：
- ✅ 首次创作新类型内容
- ✅ 技术细节复杂的文章
- ✅ 准备广泛传播的重要内容
- ✅ 用户明确要求质检

**可以跳过**：
- ⚪ 短篇快讯（<2000字）
- ⚪ 内部分享（非正式发布）
- ⚪ 时效性极高（需立即发布）
- ⚪ 同类型已多次A级

### 如何快速达到A级？

**4个高效优化**：
1. **清除AI套话**（+3-5分）
   - 搜索关键词："在当今"、"随着"、"值得注意"等
   - 用直接陈述替换

2. **标注代码语言**（+5分）
   - 所有```后加语言名称
   - 如：```python、```bash、```json

3. **拆分超长段落**（+2-3分）
   - 段落≤5行
   - 一段一个中心思想

4. **修复间距**（+1分）
   - 中英文之间加空格
   - 如：AI Agent → AI Agent

### 质检报告怎么用？

**三步走**：
1. **看总分**：确定评级（A/B/C/D）
2. **看维度**：找出得分最低的3个维度
3. **看建议**：按优先级（🔴→🟡→🟢）修复
