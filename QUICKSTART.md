## 🔧 调整配置

### 修改运行时间

编辑配置文件：
```bash
nano /data/workspace/AIContentFlow/config/aicontentflow_config.yaml
```

修改时间设置：
```yaml
schedule:
  monitor_time: "07:00"  # 改成你想要的时间
  writer_time: "08:00"   # 改成你想要的时间
```

### 修改选题数量

```yaml
monitor:
  topic_count: 5  # 改成3-10之间的数字
```

### 修改内容字数要求

```yaml
writer:
  min_words: 3000  # 最少字数
  max_words: 8000  # 最多字数
```

## ❓ 常见问题

### Q1: 如果07:00没收到通知怎么办？
检查企业微信的「Knot消息通知」机器人，或手动运行测试：
```bash
python /data/workspace/AIContentFlow/monitor/aicontentflow_monitor.py
```

### Q2: 可以临时关闭自动运行吗？
可以，联系管理员暂停定时任务。

### Q3: 选题不满意可以换吗？
可以！直接回复你想要的主题关键词，而不是数字1-5。

### Q4: 内容不满意可以修改吗？
可以！回复"需要修改"后说明修改要求即可。

### Q5: 周末会运行吗？
会！系统每天都会运行。如果周末不想运行，可以临时关闭。
